How to Play

To move:

You can move by using W,A,S,D and the arrow Keys:
W = Move upwards
A = Move left
S = Move down
D = Move right
Also you can move diagonally:
W + A = Move diagonally upwards left
W + D = Move diagonally upwards right
S + A = Move diagonally downwards left
S + D = Move diagonally downwards right
The same goes with the arrow keys

To Hit:

You can hit by clicking mouse:
Left Mouse Click = Hit
Hold Down Left Mouse Click - Keep hitting
To automatically hit press the space bar once
Do the same to disable

To Chat:

Enter - LCF (Local Chat Feature)
Enter + (Any Message) -> Enter = outputs Any message
Note: A white bubble above you should appear. This is where you type

Place Blocks:

Note: Click on the Number or Letter, then left click to Place it
1 = Place Wooden Wall
2 = Place Stone Wall
3 = Place Door
4 = Place Peasant
5 = Place Bed
6 = Place Guard
7 = Place Mercenary
8 = Place Tower
9 = Place Heart Stone

HotKeys

B = Open Item Shop (Scroll to find items)
F = Switch Weapons (Sword / Ax)
U = Dismount Ride
T = Open Teams Menu
H = Open Hats Menu
P = Open Diamond Shop

Commands:

/s (+message) = Talks in Public Server Chat (Available for only logged in users)
/help = Displays all Commands
/whereami = Displays current co-ordinates
/stats = Displays player statistics (e.g. kills, deaths)
/version = Displays current Game Version
/t (+message) = Talks in Team Chat (Only available for people in a Team)
/uptime = Displays time Server was Online (Without a Reset [Hrs:Mins])
